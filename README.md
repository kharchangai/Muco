## ⚠️ Disclaimer (Please Read)

> **Project Status: Highly Unstable (Early Stage)**

Mocu is currently in its **very early stages of development** and is not stable. Due to potential bugs and unfinished features, **it is not recommended for daily use at this time**. 
Please wait for the official Alpha release. This current version is strictly for testing and observation. Expect massive changes and frequent updates in the future.

---

## 🖱️ How to Interact with Mocu

Interacting with Mocu is designed to be simple and minimal:

*   🎙️ **Voice Recording:** **Left-click once** on Mocu to start recording your voice. Click again to stop recording.
*   ✋ **Move Around:** **Click and hold** the left mouse button on Mocu, then drag your mouse to move him anywhere on your desktop screen.

---

## 🚀 Current Features (Experimental)

Mocu currently supports the following capabilities. **Note that all of these tools are in their earliest versions and may contain significant bugs:**

- 📅 **Scheduling:** Set and manage schedules and tasks.
- 🌐 **Web Search:** Quick internet searches to find immediate answers.
- 👁️ **Desktop Vision:** Ability to look at and analyze your desktop screen.
- 🔬 **Deep Research:** In-depth research and analysis on complex topics.
- 💻 **Terminal Interaction:** Executing OS commands directly through the terminal.
- 🧠 **Memory Customization:** Remembering user information and context for personalized future interactions.

## 🛠️ Prerequisites & Installation

To use Mocu's AI capabilities, **you MUST provide your own API Key from a supported AI provider** and configure it within the app's settings.

This application is built using the **Tauri** framework. To run the source code:

### 1. Install Dependencies
```bash
npm install
```

### 2. Run in Development Mode
To run Mocu locally for development:
```bash
npm run tauri dev
```

### 3. Build the Application
To compile and build the final executable for your operating system:
```bash
npm run tauri build
```
